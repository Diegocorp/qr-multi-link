# QR Multi-Link - Microbiology Research Portal

A simple, mobile-first React + Vite single-page application designed for QR code scanning. Users land on this page after scanning a QR code and can access research articles.

## Features

- Mobile-optimized design
- Two-button interface for accessing research articles
- Microbiology-themed styling
- Links open in new tabs

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
